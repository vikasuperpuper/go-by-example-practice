# Задачи по теме: Hello World

## Задача 1: Персонализированное приветствие

Сделай программу, которая:

1. Создает переменную с именем, например `name := "Vika"`.
2. Создает переменную с любимым языком, например `lang := "Go"`.
3. Выводит строку в формате: `Привет, Vika! Добро пожаловать в Go!`

### Усложнение (по желанию)

1. Добавь переменную `day := "Friday"`.
2. Выведи вторую строку: `Сегодня Friday - отличный день учить Go.`

### Что проверяем

1. Программа запускается через `go run`.
2. Используется `fmt.Println` или `fmt.Printf`.
3. Вывод содержит подстановку переменных без ошибок.
