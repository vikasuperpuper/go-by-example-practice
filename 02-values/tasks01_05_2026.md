# Задачи по теме: Values

## Задача 1: Мини-калькулятор значений

Сделай программу, которая выводит результаты простых операций со значениями:

1. Сложение двух чисел, например `1 + 1`.
2. Конкатенацию двух строк, например `"go" + "lang"`.
3. Булево выражение, например `true && false`.

### Усложнение (по желанию)

1. Добавь еще минимум 2 своих примера со значениями разных типов.
2. Для каждого примера коротко подпиши в комментарии, какой ожидается результат.

### Что проверяем

1. В программе есть примеры с числами, строками и bool.
2. Вывод понятный, видно, какая операция к какому результату привела.
3. Программа запускается через `go run` без ошибок.
