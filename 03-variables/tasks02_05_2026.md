# Задачи по теме "Переменные" в Go

## Уровень 1: Базовые концепции

### Задача 1.1: Различные способы объявления переменных
Создай программу, которая:
- Объявляет переменную типа `string` с инициализацией через `var`
- Объявляет несколько переменных разных типов (`int`, `float64`, `bool`) одновременно
- Использует короткое объявление `:=` для переменной типа `rune`
- Объявляет переменную без инициализации (нулевое значение) и выводит её
- Покажи все эти значения в одной программе с комментариями

### Задача 1.2: Нулевые значения
Напиши программу, которая выводит нулевые значения для каждого типа:
```
string -> ""
int -> 0
float64 -> 0
bool -> false
```
Используй переменные с `var` без инициализации.

## Уровень 2: Практическое применение

### Задача 2.1: Калькулятор с переменными
Реализуй простой калькулятор, который:
- Объявляет две переменные для чисел (тип `float64`)
- Выполняет операции: +, -, *, /
- Выводит результаты каждой операции с пояснениями
- Используй как `var`, так и `:=` для разных переменных
- **Бонус**: обработай деление на ноль

### Задача 2.2: Информация о пользователе
Напиши программу, которая создает "профиль пользователя":
- Имя (string)
- Возраст (int)
- Рост в метрах (float64)
- Пол (string или bool)
- Статус активности (bool)
- Выведи всю информацию красивым форматом

## Уровень 3: Advanced

### Задача 3.1: Область видимости переменных (Scope)
Напиши программу с вложенными блоками кода:
```go
func main() {
    x := 1
    {
        x := 2  // новая переменная, скрывает внешнюю
        // какой x здесь?
    }
    // а какой x здесь?
}
```
Покажи результат и объясни, что происходит с переменными в разных блоках.

### Задача 3.2: Type inference vs explicit typing
Создай программу с несколькими примерами, где:
- Type inference помогает (явный тип был бы многословнее)
- Явный тип лучше (для ясности кода)
- Type inference может привести к ошибкам (если есть такие случаи)

### Задача 3.3: Перезапись переменных
Напиши программу, которая:
- Объявляет переменную с одним значением
- Меняет её несколько раз
- Выводит значение после каждого изменения
- Покажи, как одна переменная может менять тип? (или почему не может?)

## Уровень 4: Челлендж

### Задача 4.1: Обмен значений между переменными
Реализуй обмен значений между двумя переменными:
- БЕЗ третьей переменной (используя арифметику или XOR)
- С третьей переменной (классический способ)
- Покажи результаты обоих методов

### Задача 4.2: Определение типа по значению
Напиши программу, которая:
- Создаёт переменные с разными значениями используя `:=`
- Определяет и выводит тип каждой переменной (`fmt.Printf("%T\n", variable)`)
- Включи разные типы: целые числа, float, string, bool, рифероцинтные типы

### Задача 4.3: "Угадай значение"
Что выведет эта программа? Предскази без запуска, потом проверь:
```go
package main
import "fmt"

func main() {
    var a int = 10
    var b = a
    b = 20
    
    fmt.Println("a =", a, ", b =", b)
}
```
Объясни почему именно эти значения.
