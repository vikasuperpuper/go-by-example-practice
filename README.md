От gopherchan2006:
Ручками переписываешь примеры с сайта gobyexample.com, в своей подпапке
ты найдешь файлик с названием темы, туда и нужно переписать примерчик и запустить
пару раз, изучить его. А также ты найдешь файл формата tasks_<date>.go,
это заглушка, в этом файлие ты должна выполнить свою задачу.
А в самой теме ты найдешь файл формата tasks<date>.md, там лежит сама задача,
удачи тебе!!! :3
---
И еще момент, каждое выполненное задание ты должна комитить и отправлять в репозиторий,
это делается очень легко, например, когда ты переписала пример с gobyexample
ты пишешь git add 01-hello-world/vikasuperpuper/hello-world.go,
а затем git commit -m "hello-world", т.е пишешь название файла который добавила,
после этого git push origin main
